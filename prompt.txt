Prompt 01

Create a professional and responsive Student Toolkit web application using HTML, CSS, and JavaScript. The website should be designed for students to help manage their academic activities.

Core Requirements

Use HTML for structure, CSS for styling, and JavaScript for functionality.

Organize the layout using CSS classes and ensure a clean academic-style interface.

Layout Structure

The page must contain the following sections:

1. Navigation Bar (class: navbar)

Positioned at the top of the page.

Include the following menu items:

GPA Calculator

Study Timer

The navbar should have a professional academic look with a background color and aligned menu items.

2. Sidebar (class: sidebar)

Positioned on the left side of the page.

Include a section titled "Upcoming Events".

Display a list of example academic events such as:

Assignment Deadline

Exam Date

Study Group Meeting

3. Main Content Area
The main content area should dynamically display the following tools when selected from the navbar:

GPA Calculator

Input fields for subject names and grades.

A button to calculate GPA.

Display the calculated GPA result using JavaScript.

Study Timer

A simple study timer (start, pause, reset).

Display elapsed study time.

Styling Requirements

Use CSS classes such as:

.navbar

.sidebar

.main-content

Ensure the layout is clean and well-structured.

Use modern fonts and spacing for a professional academic appearance.

Additional Requirements

The layout should clearly separate navbar, sidebar, and main content.

Use flexbox or grid for layout alignment.

Add basic hover effects for navigation items.

Ensure the code is well commented and easy to understand for students learning web development.

Output the project as three separate files:

index.html

styles.css

script.js
