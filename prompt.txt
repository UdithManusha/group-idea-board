Design a clean and modern web interface for a collaborative tool called “Group Idea Board.” The page should let users quickly add ideas and view ideas shared by others. Keep the layout centered inside a card-style container with rounded corners and soft shadows on a light gray or subtle gradient background. At the top, include a header with the title “Group Idea Board”** using a modern sans-serif font and a subtle blue accent to make it stand out.

Below the header, add an input section with a dropdown to select a name, a text field to type an idea, and a clear “Add Idea” button, arranged in one row with good spacing. Under this area, display submitted ideas in simple card-style list items showing the idea text and a small label like “suggested by [name]”. Keep the design minimal, responsive, and similar to modern productivity apps, with soft hover effects and clean spacing.
