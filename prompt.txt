Create a modern and visually appealing web interface for a small collaborative tool called “Group Idea Board.” This page should allow users to quickly add ideas and see ideas suggested by others in the group.

The layout should feel clean, organized, and centered on the page, similar to modern productivity tools. Place everything inside a card-style container with soft shadows and rounded corners so it stands out from a light gray or subtle gradient background.

At the top of the card, add a header section with the title “Group Idea Board.” The header can have a soft blue gradient or subtle color bar to make it visually distinct. Use a modern sans-serif font and make the title clear and bold.

Below the header, create a simple input section where users can submit new ideas. This section should contain:

A dropdown menu where users can select their name (placeholder text: “Select your name…”).

A text input field where they can type their idea (placeholder: “Enter your idea here…”).

A clearly visible “Add Idea” button.

Arrange these elements horizontally in one row, with comfortable spacing between them. The button should look like a primary action button, preferably in a blue tone with a subtle hover effect.

Under the input area, display a list of ideas that have already been suggested. Instead of simple bullet points, show each idea in a clean list item or small card style. Each item should display:

The idea text

A smaller line that says “suggested by [name]”

Use good spacing, soft dividers, and subtle hover effects so the list feels interactive and easy to read.

For the overall design, aim for something that looks similar to modern SaaS dashboards or productivity apps. Keep the design minimal, balanced, and user-friendly, with consistent spacing, smooth edges, and a small amount of color for visual focus.

Optionally, you can enhance the interface with small touches such as:

Hover effects for buttons and list items

Small user avatars next to names

Icons for editing or removing ideas

Smooth animations when a new idea is added

A responsive layout that also works well on mobile screens

The final design should feel professional, modern, and intuitive, making it easy for a group to quickly share and view ideas.
